# instagram-id-of

> Prints the userid of facebook user in your console.

## Install

```
$ npm install --global facebook-id-of
```

## Usage

```
$ facebook-id-of -u '/user.name'

  Usage :

    $ insta-id-of -u '/zuck'

    Status Code : 200

    Facebook Userid of ZUCK is : 4

```
